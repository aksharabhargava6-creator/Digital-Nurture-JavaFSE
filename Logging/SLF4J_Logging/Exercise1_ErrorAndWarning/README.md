# Exercise 1 - Error and Warning Logging using SLF4J

## Objective
Demonstrate logging of warning and error messages using the SLF4J logging framework.

## Technologies Used
- Java
- Maven
- SLF4J
- Logback

## Project Structure

```
Exercise1_ErrorAndWarning
│
├── pom.xml
├── README.md
├── Output.png
└── src
    └── main
        └── java
            └── LoggingExample.java
```

## Code Description
The application uses SLF4J with Logback as the logging implementation.

It demonstrates:
- ERROR level logging
- WARN level logging

## Sample Output

```
ERROR LoggingExample - This is an error message
WARN  LoggingExample - This is a warning message
```

## Result
Successfully implemented error and warning logging using the SLF4J framework.
