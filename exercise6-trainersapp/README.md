# Exercise 6 - Trainers App

## Objective

Build a React application that demonstrates routing using React Router.

## Features

- Home page
- Trainers List page
- Trainer Details page
- Navigation using React Router
- Dynamic route parameters

## Technologies Used

- React
- React Router DOM
- JavaScript

## Folder Structure

```
src
│── Components
│   ├── Home.js
│   ├── TrainersList.js
│   └── TrainerDetails.js
│
├── trainer.js
├── TrainersMock.js
├── App.js
└── index.js
```

## Output

- Home page displays welcome message.
- Trainers page displays all trainers.
- Clicking a trainer opens trainer details.
