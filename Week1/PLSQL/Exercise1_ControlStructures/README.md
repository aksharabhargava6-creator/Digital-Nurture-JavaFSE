# Exercise 1: Control Structures

## Aim
Demonstrate PL/SQL control structures using loops and conditional statements.

## Description
This exercise uses a FOR LOOP to iterate through employee records and an IF condition to display employees whose salary is greater than 5000.

## Code Features
- FOR LOOP
- IF Statement
- DBMS_OUTPUT.PUT_LINE
- HR.EMPLOYEES table

## Output
Displays Employee ID and Salary for employees having salary greater than 5000.

## Complexity
Time Complexity: O(n)
Space Complexity: O(1)
