# Exercise 2 - Student Management Portal

## Objective

Create a React application using Class Components to display the Home, About, and Contact pages of a Student Management Portal.

## Tools Used

- React
- Node.js
- npm
- Visual Studio Code

## Components Created

- Home
- About
- Contact

## Steps Performed

1. Created a React project using Create React App.
2. Created a `Components` folder.
3. Implemented three class components:
   - Home
   - About
   - Contact
4. Imported all components into `App.js`.
5. Rendered all components on the webpage.
6. Executed the application using:

```bash
npm start
```

## Output

The application successfully displays:

- Welcome to the Home page of Student Management Portal
- Welcome to the About page of Student Management Portal
- Welcome to the Contact page of Student Management Portal
