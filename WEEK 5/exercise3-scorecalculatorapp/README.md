# Exercise 3 – Score Calculator App

## Objective

Build a React application that displays student details and calculates the percentage score using reusable React components.

## Features

- Display student name
- Display school name
- Display total marks
- Calculate percentage score
- Apply CSS styling using an external stylesheet

## Technologies Used

- React
- JavaScript (ES6)
- CSS
- Create React App

## Project Structure

```
scorecalculatorapp/
│── src/
│   ├── Components/
│   │   └── CalculateScore.js
│   ├── Stylesheets/
│   │   └── mystyle.css
│   ├── App.js
│   └── index.js
│
├── Output.png
├── OutputTerminal.png
├── package.json
└── README.md
```

## Output

The application displays:

- Student Name
- School Name
- Total Marks
- Percentage Score
